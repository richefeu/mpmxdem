\subsection{Node grid}

In MPMbox, the fixe grid can only be regular. It can be set with the command  \syntax{\keyword{set\_node\_grid}} to allocate the nodes. Depending on the token that follows the command, the parameters used to set the grid differ. The options are:  \syntax{\keyword{Nx.Ny.lx.ly}}, \syntax{\keyword{W.H.lx.ly}} or \syntax{\keyword{W.H.Nx.Ny}}. Note that the bottom-left corner is placed at the origine $O(0,\, 0)$.\\

\subsection{Constitutive Models}


Before using a constitutive model, you must first create one by assigning it an identifier, specifying the model type, and providing the corresponding parameters.

\begin{codeblock}
\syntax{\keyword{model} \type{(string)}\param{modelIdentifier} \type{(string)}\param{modelName} (...list of parameters...)}
\end{codeblock}

Each material point will then be associated with the model through its identifier.
Here are the lists of parameters for each \syntax{\param{modelName}}:
\begin{itemize}
\item  \syntax{\keyword{HookeElasticity}}: \syntax{\type{(double)}\param{Young} \type{(double)}\param{Poisson}}
\item  \syntax{\keyword{VonMisesElastoPlasticity}}: \syntax{\type{(double)}\param{Young} \type{(double)}\param{Poisson} \type{(double)}\param{PlasticYieldStress}}
\item  \syntax{\keyword{MohrCoulomb}}: \syntax{\type{(double)}\param{Young} \type{(double)}\param{Poisson} \type{(double)}\param{FrictionAngle}}\\ 
\syntax{\type{(double)}\param{Cohesion} \type{(double)}\param{DilatancyAngle}}
\item  \syntax{\keyword{CHCL\_DEM}}: \syntax{\type{(string)}\param{conf-filename}}
\end{itemize}


\subsection{Adding material points}


Material points can be added by placing them on a grid. The corresponding command in this case is: \\

\begin{codeblock}
 \syntax{\keyword{set\_MP\_grid} \type{(int)}\param{group} \type{(string)}\param{modelIdentifier} \type{(double)}\param{$\rho$} \type{(double)}\param{$x_0$} \type{(double)}\param{$y_0$} \type{(double)}\param{$x_1$} \type{(double)}\param{$y_1$}}
\label{setMPgrid}
\end{codeblock} 
where ($x_0,y_0$) denotes the coordinates of the beginning corner of the material points square grid and the opposite one is at ($x_1, y_1$). $\rho$ signifies the density of each material point and must be specified, except in the \texttt{CHCL\_DEM} model where it gets calculated automatically via the DEM configuration:

\begin{equation}
  \rho_{\text{mp}} = \dfrac{m_{\text{VER}}}{V_{\text{VER}}} = \dfrac{\det(h)}{V_s}
  \label{}
\end{equation}
which the diagonal matrix \textit{h} denotes the periodic boundary domain.

\subsection{Discrete Element Method}
\subsubsection{Isotropic Compression}
Firstly, a sample of DEM is prapared by isotropic compression in order to: i) Serve as an VER which will be be integrated in every material points for the MPMxDEM stage. ii) The triaxial simulation will be conducted on this sample, to obtain the paramaters of the constitutive law Mohr Coulomn in MPM.
Initial input parameters are shown in \cref{demParam} which satisfied the: i) Critical time step $dt <= dt_c =  2\pi \sqrt{m / k_n}$ ii) stiffness level $\kappa = k_n / (a \sigma_0) = 1000$

\subsubsection{Triaxial Compression}
$I$ is chosen via the velocity of deformation to be lower than $10^{-4}$ for quasi-static response.
Thus the necessary parameters for MC model: $E$, $\nu$, $\varphi$, $c$, $\psi$ could be indentified. Based on the deviatoric stress peak of both compression and extensionâ€™s process, $\varphi = 35^\circ$ and $c = 0$, which corresponds to the initial configuration of dry material.
Module Young is found to be 33 Mpa (\cref{triaxStress}), and based on the tangent of compression and dilation stage once could get $\nu$ and $\psi$ respectively.
